# Legal Notes

MineForgeAI Omniverse does not ship Minecraft source, jars, assets, mappings, or decompiled code.

- Minecraft jars, assets, and version metadata remain subject to Mojang and platform terms.
- Decompiled source is blocked unless the user explicitly confirms the action.
- The project stores only safe summaries, indexes, metadata, and user-generated outputs.
- The web layer blocks piracy, cracked plugins, malware, hacked clients, and credential exfiltration.
- The agent never uploads user code or jars without explicit user permission.
